# Hard Pass

A simple app to help you say no.

Tap → get a response → copy or send → move on.

---

## Purpose

Hard Pass reduces the friction of saying no.

It gives you short, natural responses you can actually use in real life—whether you need something polite or just want something funny.

This is not a productivity app.  
This is a **communication shortcut**.

---

## Core Experience

1. Open app  
2. Tap “Give Me an Excuse”  
3. See response  
4. Copy or share  

That’s it.

---

## Features (V1)

- One-screen interface  
- Random response generator  
- Copy to clipboard  
- Native share (text, email, etc.)  
- Curated response list (not AI-generated)

---

## Tone Categories

V1 will include:

- **Polite** → usable in real situations  
- **Funny** → light / shareable  

Future (not in V1):
- Firm  
- Sarcastic  

---

## Product Principles

- Fast (no friction)  
- Simple (no setup, no accounts)  
- Natural (responses must sound real)  
- Minimal (no unnecessary features)

---

## Out of Scope (V1)

- Accounts / login  
- Saving history  
- Custom responses  
- AI-generated responses  
- Settings menus  

---

## Definition of Done (V1)

- Button generates a random response  
- Response displays clearly  
- Copy button works  
- Share function works
